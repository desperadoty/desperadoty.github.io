mysqlclient安装与配置

发表于 2019-10-12 | 分类于 DataBase

brew install mysql-connector-c

pip3 install mysqlclient

以下报错原因：升级了macOS Catalina之后，原来xcode也需要更新

cd /usr/local/Cellar/mysql-connector-c/6.1.11/bin

sudo chmod -R 777 mysql_config

vi mysql_config

需要修改的内容：
libs="-L$pkglibdir"
libs="$libs -l"

替换为：
libs="-L$pkglibdir"
libs="$libs -lmysqlclient -lssl -lcrypto"

重新安装mysqlclient

Android APP性能测试基础

查看CPU占用率的两条命令：

①adb shell top -n 1

②adb shell dumpsys cpuinfo | grep pkgname

基本数据存储位置

adb shell
cd /proc/$pid
ls

查看程序的内存使用情况:

adb shell dumpsys meminfo pkgname

VSS - Virtual Set Size 虚拟耗用内存
RSS - Resident Set Size 实际物理内存
PSS - Proportional Set Size 比例分配物理内存（统计内存大小）
USS - Unique Set Size 进程独占内存（测试内存泄露）

一般来说，以上四个数据依次递减

帧率，最直接的影响是界面的流畅度

adb shell dumpsys gfxinfo pkgname

安卓⼿机屏幕的最⾼高帧率是60帧/秒， 那么为了保持这个帧率，一个界⾯面的渲染时间不不能超过1/60秒,即16.7ms

计算方法：
实时帧率 = 1 / ⼀个界⾯渲染所花费的时间
帧率 = 60-(单位时间内所有问题帧的耗时/16.7)

Jmeter Plugins - Once Only Controller

发表于 2019-06-18 | 分类于 Jmeter

顾名思义，Once Only Controller在每个线程组仅执行一次。

一般将它作为Loop Controller的子节点，常常应用于用户登录场景。

eg.设置所有请求循环两次，但对登录请求添加Once Only Controller。


执行测试计划并查看结果，可见登录请求仅执行一次，其他请求执行两次。

MySQL数据库基本操作

Posted on 2017-10-05

数据库操作

1.创建数据库
mysql>CREATE DATABASE test; //创建test数据库

2.查看数据库
mysql>SHOW DATABASES; //查看所有数据库

3.选择数据库
mysql>USE DATABASE test; //使test成为当前默认数据库

4.删除数据库
mysql>DROP DATABASE test; //删掉test数据库

5.备份数据库

mysqldump -u root -p test > /home/bak.sql //备份test数据库
mysqldump -u root -p --databases db1 db2 db3 > /home/bak.sql //备份多个数据库
mysqldump -u root -p -all-databases > /home/bak.sql //备份所有数据库
cat bak.sql //查看备份文件是否与原来数据库一致

6.还原数据库
mysql -u root -p test < /home/bak.sql //执行备份文件进行恢复
